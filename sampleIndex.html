Simple Chatbot Widget Demo

This is a minimal example of the chatbot widget integration

Quick Start

Copy and paste this code into your HTML file:

<!-- Add React and ReactDOM -->
<script src="https://unpkg.com/react@18/umd/react.production.min.js"></script>
<script src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js"></script>

<!-- Add the widget script -->
<script src="dist/chatbot-widget.umd.js"></script>

<!-- Initialize the widget -->
<script>
  ChatbotWidget.init({
    apiHost: 'http://localhost:3000',
    chatflowid: 'demo-chatflow'
  });
</script>

Try it out!

The chatbot widget should appear in the bottom right corner of this page.

You can customize the appearance by adding theme options:

ChatbotWidget.init({
  apiHost: 'http://localhost:3000',
  chatflowid: 'demo-chatflow',
  theme: {
    chatWindow: {
      height: 600,
      width: 400
    }
  }
});
